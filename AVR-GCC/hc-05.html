Módulo Bluetooth HC-05

Aprende a configurar y utilizar el módulo HC-05 mediante comandos AT para cambiar nombre, contraseña y comunicación de datos

Introducción al HC-05

¿Qué es el HC-05?

  • Módulo Bluetooth de bajo costo
  • Compatible con dispositivos Android/iOS
  • Funciona en modo esclavo y maestro
  • Comunicación serie UART

Características principales

  • Versión Bluetooth 2.0 + EDR
  • Alcance: ~10 metros
  • Voltaje: 3.3V - 5V
  • Velocidad: 9600 bps por defecto
Ideal para proyectos IoT y control remoto

Comandos AT Básicos

Modo Comando AT

  • Presionar botón antes de conectar
  • LED parpadea lentamente (2 veces/seg)
  • Velocidad: 38400 bps
  • Formato: AT+COMANDO

Comandos esenciales

  • AT - Comando de prueba
  • AT+VERSION - Consultar versión
  • AT+NAME? - Consultar nombre
  • AT+UART? - Consultar parámetros UART
Respuesta OK indica comando exitoso

Cambiar Nombre

Comando para nombre

  • AT+NAME=nuevo_nombre
  • Máximo 20 caracteres
  • Sin espacios ni caracteres especiales
  • Ejemplo: AT+NAME=MiHC05

Verificar cambio

  • AT+NAME? - Consultar nombre actual
  • Respuesta: +NAME:nombre_actual
  • Reiniciar módulo para aplicar cambios
AT+NAME=RobotControl
OK

AT+NAME?
+NAME:RobotControl
OK
El nombre aparece al buscar dispositivos Bluetooth

Configurar Contraseña

Comando para PIN

  • AT+PSWD=contraseña
  • Por defecto: 1234
  • Máximo 4 dígitos numéricos
  • Ejemplo: AT+PSWD=5678

Verificar PIN

  • AT+PSWD? - Consultar contraseña
  • Respuesta: +PSWD:contraseña
  • Se requiere al emparejar
AT+PSWD=9999
OK

AT+PSWD?
+PSWD:9999
OK
Cambia la contraseña por seguridad

Configuración UART

Parámetros de comunicación

  • AT+UART=baud,stop,parity
  • baud: 9600, 19200, 38400, etc.
  • stop: 0=1 bit, 1=2 bits
  • parity: 0=ninguna, 1=impar, 2=par

Ejemplos comunes

  • AT+UART=9600,0,0 - 9600 bps, 1 stop, sin paridad
  • AT+UART=115200,0,0 - Alta velocidad
  • AT+UART? - Consultar configuración
AT+UART=115200,0,0
OK

AT+UART?
+UART:115200,0,0
OK
Velocidad debe coincidir en ambos lados

Enviar y Recibir Datos

Modo Transparente

  • Sin botón presionado al conectar
  • LED parpadea rápido (1 vez/seg)
  • Datos se envían directamente
  • Sin formato AT+

Ejemplo de uso

  • Conectar desde app móvil
  • Enviar datos como texto plano
  • Microcontrolador procesa datos
  • Respuesta automática
Comunicación bidireccional en tiempo real

Modo Maestro/Esclavo

Configurar rol

  • AT+ROLE=0 - Modo esclavo
  • AT+ROLE=1 - Modo maestro
  • AT+ROLE=2 - Modo esclavo-loop
  • Por defecto: 0 (esclavo)

Emparejamiento maestro

  • AT+BIND= dirección_MAC
  • AT+CMODE=0 - Conexión fija
  • AT+INQM=1,9,48 - Búsqueda
  • AT+INIT - Inicializar perfil
Modo maestro permite conexiones automáticas

Reset y Restauración

Comandos de gestión

  • AT+RESET - Reiniciar módulo
  • AT+ORGL - Restaurar valores de fábrica
  • AT+ADDR? - Consultar dirección MAC
  • AT+STATE? - Estado de conexión

Solución de problemas

  • No responde: Verificar modo AT
  • Sin conexión: Revisar velocidad
  • Error: Usar AT+ORGL
  • Sin datos: Verificar emparejamiento
AT+ORGL restablece nombre y PIN por defecto
